# CRM Comparison - UCX vs v6

Landing page moderna para comparar **uc-uCRM (UCX)** y **uc-CRM-Omni (v6)**.

## 🚀 Inicio Rápido

### Instalación

```bash
npm install
```

### Desarrollo

```bash
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000) en tu navegador.

### Build para Producción

```bash
npm run build
npm start
```

## 🎨 Características

- ✅ Diseño moderno tipo SaaS (inspirado en Stripe, Vercel, Linear)
- ✅ Responsive (mobile, tablet, desktop)
- ✅ Dark mode automático
- ✅ Navegación por tabs
- ✅ Comparativa visual con badges de estado
- ✅ Iconos Lucide React
- ✅ Tailwind CSS
- ✅ Next.js 14 + TypeScript

## 📋 Secciones

1. **Hero** - Presentación principal
2. **Resumen** - Vista general de ambos sistemas
3. **UCX** - Características de uc-uCRM
4. **v6** - Características de uc-CRM-Omni
5. **Comparativa** - Tabla detallada lado a lado
